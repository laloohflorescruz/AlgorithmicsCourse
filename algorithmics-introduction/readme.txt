This folder contains the demo sources for the Pluralsight course "Introduction to Algorithmics".

There are two root level folders:

   * LogReading/ holds an XCode project with the iPhone app that is demonstrated in the first module.

   * IntroductionToAlgorithms/ holds a Visual Studio 2015 solution with all the C# code from the demos,
     all the source code snippits that have been shown in the presentations, and implementations of the
     algorithms described in Module 4.

REQUIREMENTS
The source code depends on the C5 NuGet package.
